Process for downloading website templates

There are major three types of process by which you can download paid and unpaid templates easily :

  • Direct Download
  • Httrack
  • Resource saver

Below mentioned the detailed process.




1. Direct Download


  • First, you have to visit bootstrapmade.com website which provide free website templates. Now select any template also you can see live demo here.

  • After preview now click on free download button which appear on top-right corner.

  • Now Click on this free download button.

  • Your template starts downloading in few seconds.

    • 2. Httrack


  • First, you have to visit colorlib.com website which provide paid website templates. Now select any template also you can see live demo here.

  • After preview now copy the link of address bar.

  • Now httrack software and click on next button.

  • Fill all the fields and click on next button.

  • Paste or enter the copied url here.

  • Click on finish button.

  • Finally, your downloading finished.

    • 3. Resources Saver


  • First, you have to search save all resources chrome extension in google tab.

  • Then various links will be open and you have to click on topmost google reference link.

  • Click on Add to chrome button.

  • Click on Add Extension button and after few seconds it adds the extension and appears a button named remove from chrome.

  • Now, you have to visit themeforest.com website which provide paid website templates. Now select any template also you can see live demo here.

  • After preview now right click on template and select inspect.

  • Now Click on this resource saver button and click all check-boxes. After few seconds on the right corner the save all resources will be appears.

  • Click on save all resources button then your template starts downloading in few seconds.
